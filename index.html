Downloads

    Critcl currently has two major branches, version 2, and version 3.

    The head of v2 is version 2.1, to be found at the same-named tag in the repository. This version is compatible with existing installations of version 2, as it only has some bug fixes and minor, API-compatible extensions. For details see the full list of Changes for version 2.1.

    Version 3 on the other hand, has two API incompatibilities against version 2, beyond more bug fixes and major new features, necessitating the major version change. For details see the full list of Changes for version 3.

    It is also the only version of Critcl which is under active development.
