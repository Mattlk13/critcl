critcl by jcw

Critcl lets you easily embed C code in Tcl.

For more background see http://equi4.com/starkit/critcl.html

 

Dependencies

  • Tcl
  • platform (part of newer Tcl distributions)
  • md5
  • cmdline
  • md5c (accelerator for md5)
  • Trf (alternate accelerator for md5)

The repository contains copies of the sources of most of these dependencies. The exceptions are Tcl itself, and Trf.

Install

Just run

$ ./build.tcl install

This is the most simple form. More forms are explained in the README.md.
